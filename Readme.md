udacity-ripper
==============

*udacity-ripper* is a python script that uses Selenium to crawl Udacity
course pages to extract all the lesson YouTube links.

## Prerequisites

* *Python* `>= 2.7`
* *Selenium* `>= 3.8.0`

# Usage

```bash
pip install selenium
python udacity-ripper.py <CLASS URL> <LOGIN> <PASSWORD>
```
